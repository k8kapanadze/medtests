# MedCore — Specialized Medical Testing Platform

Professional adaptive medical testing platform built for medical students and residents.

## Features

- Adaptive mistake-focused learning
- Double shuffle engine
- Mistake review mode
- Flagged question collections
- Multi-file mini exam simulator
- Dark / Light mode
- Keyboard shortcuts
- Mobile-first responsive UI
- Autosave and resume system

## Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- Framer Motion

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vision

The platform is designed to help students eliminate weak topics faster and master massive medical question banks with maximum efficiency.
